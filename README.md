# IMDB Data Analysis

This project analyzes the IMDB Top 1000 movies dataset using Python.

## Features

* Top rated movies
* Genre analysis
* Rating distribution
* Votes vs Rating regression
* Director analysis
* Year-wise trends

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## How to Run

1. Download files
2. Run:
   python imdb_analysis.py

## Output

Graphs showing movie insights and trends.
